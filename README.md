# 🏥 Sistema de Fila Hospitalar

Projeto desenvolvido na disciplina lógica de programação com o objetivo de simular um sistema de gerenciamento de fila hospitalar.

## Objetivo

Organizar o atendimento de pacientes por meio de uma fila, registrando e controlando a ordem de atendimento.

## Funcionalidades

- Cadastro de pacientes
- Organização da fila de atendimento
- Registro de informações em arquivo JSON
- Controle da persistência dos dados

## Tecnologias utilizadas

- 🐍 Python
- 📄 JSON

## Como executar

No terminal:

```bash
streamlit run app_web.py
